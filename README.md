# homelab-devops

Homelab portfolio for DevOps / Linux / Infrastructure practice.

## Goals
- Build reproducible Linux-based environment
- Run containerized services (Docker/Compose)
- Implement CI/CD deployments
- Add monitoring/alerting and logging
- Document runbooks and incident reports (RCA)

## Structure
- `docs/` — architecture, notes
- `runbooks/` — operational procedures (deploy/rollback/troubleshooting)
- `incidents/` — incident reports (symptom → RCA → fix → prevention)
- `diagrams/` — diagrams (Mermaid/PNG)
- `projects/` — project-specific configs and docs
